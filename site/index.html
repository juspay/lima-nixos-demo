devbox
GitHub Lima Start Script

NixOS devbox for local development.

A prebuilt NixOS VM image that runs on Lima. Use it as a disposable, repeatable development machine without mounting your macOS home directory.

  • Prebuilt: boots from GitHub release images.
  • Practical: includes flakes, direnv, VS Code server, gh, just, and common tools.
  • Contained: exposes only /tmp/lima-devbox for intentional file transfer.

Start In One Command

Install Lima first. On macOS, run brew install lima.

curl -fsSL https://juspay.github.io/devbox/start | sh -
The starter reuses a running VM, starts a stopped VM, or creates one from the latest image.
Open a shell: limactl shell --workdir=. devbox
Customize: curl -fsSL https://juspay.github.io/devbox/start | sh -s -- --help
